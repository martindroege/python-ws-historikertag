User-Input Skript

# Eingabe Name
name = input("Wie heißt du? >>>")
# Eingabe Alter
alter = input("Wie alt bist du? >>>")
# Eingabe Wohnort
ort = input("Wo wohnst du? >>>")
jahr = 2018 - int(alter)
# Ausgabe
print(f"""
\nHallo {name}, schön, dass du da bist!\n
Du bist {jahr} geboren.\n
{ort} ist der beste Ort auf dem Planet.""")

Kontrollstrukturen

For-Schleife

for x in range(10):
    print(x)

While-Schleife

x = 10
while x > 0:
    print(x)
    x -= 1

If-Anweisung

for x in range(1, 11):
    if x % 2 == 0:
        print(f"{x} ist eine gerade Zahl.")
    else:
        print(f"{x} ist eine ungerade Zahl.")

I/O Parameter

  • “r” = lesen
  • “w” = schreiben
  • “a” = append / anhängen
